Welcome to comb-model's documentation!
===================================

**comb-model** (/lu'make/) is a Python library for simulation of trajectories of the comb model via Langevin equations driven by fractional Gaussian noises (long-range correlated).


Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
